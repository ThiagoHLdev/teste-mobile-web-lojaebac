
# Teste Mobile Web Loja EBAC - Appium + WebdriverIO

## Como rodar

1. Tenha o Android Emulator rodando no Android Studio
2. Tenha o Appium Server rodando
3. Tenha o Chrome instalado no emulador
4. Execute:
```
npm install
npx wdio run wdio.conf.js
```
